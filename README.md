https://random-cat-chi-gules.vercel.app/

# Dependencies
- Node.js v21 (21.2.0) 
- Yarn v1 (1.22.21)

インストールされているか確かめる方法。

バージョンが表示されていたらインストール済み。

```terminal
$ node -v
v21.2.0

$ yarn -v
1.22.21
```

# Setup
## 1/2 Clone
```terminal
$ git clone

$ cd random-cat
```

## 2/2 Start
```terminal
$ yarn dev
```
Go to http://localhost:3000/
